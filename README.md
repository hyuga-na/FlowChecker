# FlowChecker

## 概要

FlowChecker は、研究やプレゼンの論理構成（背景・課題・解法・結果・考察）を  
5行で入力し、「論理の飛躍」を検出する軽量Webツールです。

---

## 特徴

- 入力は5行のみ（シンプル）
- 即時に論理チェック
- 完全無料
- サーバ不要（ブラウザのみで動作）

---

## 使い方

1. `index.html` を開く
2. 5つの項目を入力
3. 「チェック」ボタンを押す
4. 各行の論理状態が表示される

---

## 入力形式

1. 背景
2. 課題
3. 解法
4. 結果
5. 考察

---

## 出力

- 問題なし
- 飛躍

---

## 判定ロジック（MVP）

- 前の行との単語重複をベースに簡易判定
- 空欄は「飛躍」
- 1行目は常に「問題なし」

---

## 公開方法（無料）

### GitHub Pages

1. リポジトリ作成
2. `index.html` を配置
3. Settings → Pages → Deploy

---

### Netlify / Vercel

1. フォルダをアップロード
2. 即公開
